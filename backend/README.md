# QR Attendance System - Backend

Backend API for QR Code-based attendance management system built with Node.js, Express, and MongoDB.

## 🚀 Features

- JWT-based authentication
- Role-based access control (Student/Lecturer)
- QR code generation for attendance sessions
- Automatic attendance marking
- Session expiry management
- Attendance history and analytics

## 📋 Prerequisites

- Node.js (v14 or higher)
- MongoDB (v4.4 or higher)
- npm or yarn
